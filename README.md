Simple implementation of social network maked on Java Spring Boot with using microservice technology and include Eureka, Redis, Kafka, working with AWS S3 and third party EC2 images.

Simple start:
1)
  You can find two files, .env example and /.config/common.properties example.
  Remowe from title "example".
  Fill in the blank fields.
2)
  In terminal open directory Social-network.
  Give access for build.sh (chmod +x build.sh).
  Start build.sh for comlipe all microservices.
3)
  Start in terminal "docker-compose up"

All done,
Now you can try do any request from Postgres.

Also you can try open Live Server on https://kostotestproject.it.com/
